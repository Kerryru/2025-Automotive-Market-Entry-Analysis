项目背景：
    在中国，汽车市场是全球规模最大、增长动能最强的核心消费市场，也是引领全球汽车产业技术革新与结构转型的关键阵地。近年来，随着居民人均可支配收入提升、城乡出行需求升级，以及国家 “双碳” 战略与新能源汽车产业扶持政策的持续推进，市场呈现出鲜明的变革特征,结束了过去汽车巨头垄断的局面，越来越多的新兴企业和品牌出现。

分析目的：
    我们是一家新成立的科技公司，没有传统燃油车的包袱，也没有历史品牌的桎梏。我们拥有雄厚的初期资本、一支顶尖的技术团队，以及互联网化的用户运营思维。
    我们希望在巨头林立的汽车红海中，撕开一道口子，用科技重新定义智能出行体验。计划在26年推出首款产品，让市场和用户记住我们、选择我们。

核心需求详解：
    1、市场切入需求
    作为一个后进场者，必须要在一开始就找到一个 “竞争格局有利且增长潜力明确” 的细分市场作为突破口。
    具体问题：
      1、哪个细分市场（燃油车/新能源）？
      2、哪种车型品类（如轿车、SUV）？
      3、哪个价格区间（如20-30万）既有足够的市场容量，又尚未被巨头完全垄断？
    
    2、产品定义需求
    在巨头林立的市场中，平庸等于失败。我们的产品必须有一个尖锐无比的“价值锚点”，让用户一眼记住、并产生购买冲动。即找到属于我们的“唯一性”。
    具体问题：
      1、追求性价比还是高端市场？
      2、追求实用还是外观？
      3、追求技术与性能？
      3、追求某一细分用户需求？
    
    3、竞争策略需求
    现在市场的营销价格越来越高，我们的营销资源有限，必须聪明地“借势”并精准地“攻击”。
    具体问题：
      1、目标客户地域
      2、对标竞争车型
      3、合作对象



数据集介绍
    依靠网络爬虫，从官方网站（CAAM）和头部买卖车网站（懂车帝等）获取。


数据文件详解
    共计12份文件
    共计约21000条数据
    覆盖时间从20年1月到25年9月
    共计37个独立字段，覆盖市场、品牌、车型、技术、政策、时间、销量等维度

数据维度详解
    时间维度：年度、月度
    品牌维度：1、全品牌120多个
    	       2、品牌国别（中外）
    车类型维度：1、大品类（燃油车\新能源）
    	          2、细分品类（轿车\SUV\MPV）	          3、新能源类型（纯电\混动）
    	          4、具体车型（如小米SU7）
    价格维度： 1、价格带（如10万下\10万+）
      	       2、具体报价
    区域维度：省份
    技术维度：1、续航\油耗
    	      2、智驾芯片算力
    	      3、油耗\快充时间
